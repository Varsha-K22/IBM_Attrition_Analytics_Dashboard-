📊 IBM Attrition Dashboard

An interactive Excel Dashboard Project focused on analyzing employee attrition patterns within IBM.
This project provides valuable HR insights into workforce turnover, employee experience levels, salary impact, and departmental attrition trends using dynamic Excel visualizations.

📌 Project Overview

The IBM Attrition Dashboard helps HR teams and business analysts understand the major factors influencing employee attrition.

The dashboard transforms raw HR data into actionable insights through:

- KPI Metrics
- Interactive Filters
- Visual Analytics
- Trend Analysis
- Department & Salary Insights

Built entirely in Microsoft Excel, this dashboard demonstrates strong skills in:

- Data Cleaning
- Data Visualization
- Dashboard Design
- HR Analytics
- Pivot Tables & Charts
- Interactive Reporting

📊 Dashboard Insights

🔹 Overall Attrition Analysis

Tracks annual employee attrition rates across the organization.

🔹 Experience-Based Attrition

Identifies higher attrition among employees with 0–2 years of experience.

🔹 Department-Wise Attrition

Highlights departments with significant employee turnover, especially Sales.

🔹 Salary Impact Analysis

Analyzes attrition patterns based on employee salary ranges.

🔹 Interactive Filtering

Users can dynamically filter data for deeper analysis and better decision-making.

🎯 Business Insights

Key findings from the analysis:

- Employees with 0–2 years experience show the highest attrition rate.
- The Sales department experiences higher turnover compared to other departments.
- Employees in the lower salary range (<5K) are more likely to leave.
- Attrition trends can help HR improve retention strategies.
